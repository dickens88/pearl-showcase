# OUR PHILOSOPHY 图片管理说明

## 功能说明

现在您可以在管理后台轻松添加或替换首页 "OUR PHILOSOPHY"（品牌介绍）部分右侧的图片了！

## 如何使用

### 1. 登录管理后台
访问：`http://localhost:5173/admin/login`

### 2. 进入内容编辑器  
在管理后台侧边栏，点击 **"内容编辑"**

### 3. 选择首页
- 左侧页面列表会显示"首页"选项（这是新添加的）
- 点击"首页"即可编辑首页内容

### 4. 上传 OUR PHILOSOPHY 图片
- 您会看到"OUR PHILOSOPHY 图片"字段
- 建议尺寸：**800×1000px** (竖版图片效果最佳)
- 支持格式：JPG、PNG、WebP

操作步骤：
1. 点击"选择文件"按钮
2. 选择您的图片
3. 等待上传完成（会显示"上传中..."）
4. 上传成功后会显示预览图
5. 点击右上角的"保存"按钮

### 5. 替换已有图片
如果您想更换图片：
1. 点击预览图右上角的"删除图片"按钮
2. 重新上传新图片
3. 点击"保存"

### 6. 查看效果
- 回到首页：`http://localhost:5173`
- 滚动到 "OUR PHILOSOPHY" 部分
- 您刚上传的图片会显示在右侧

## 技术实现

- **前端**：React + Vite
- **后端**：Flask API
- **数据存储**：页面内容存储在数据库中
- **图片存储**：上传的图片保存到服务器 `/uploads` 目录

## 注意事项

- 图片会自动适应容器大小
- 如果没有上传图片，会显示默认的渐变背景
- 建议使用高质量的图片以获得最佳展示效果
- 图片文件大小建议不超过 5MB

## 故障排除

**如果图片没有显示：**
1. 检查浏览器控制台是否有错误信息
2. 确认图片已成功上传（查看预览）
3. 确认已点击"保存"按钮
4. 刷新首页

**如果上传失败：**
1. 检查图片格式是否支持
2. 检查图片文件大小
3. 检查网络连接
4. 查看服务器日志
