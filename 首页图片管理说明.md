# 首页图片管理说明

## 功能说明

现在您可以在管理后台轻松添加或替换首页的两个重要区域的图片了：
1. **OUR PHILOSOPHY** (品牌介绍) 部分右侧的图片
2. **PEARL STORY** (珍珠知识) 部分右侧的图片

## 如何使用

### 1. 登录管理后台
访问：`http://localhost:5173/admin/login`

### 2. 进入内容编辑器  
在管理后台侧边栏，点击 **"内容编辑"**

### 3. 选择首页
- 左侧页面列表会显示"首页"选项
- 点击"首页"即可编辑首页内容
- 您会看到两个图片字段：
  - **OUR PHILOSOPHY 图片**
  - **PEARL STORY 图片**

### 4. 上传图片

#### 4.1 OUR PHILOSOPHY 图片
- 位置：首页"品牌介绍"部分右侧
- 建议尺寸：**800×1000px** (竖版图片)
- 支持格式：JPG、PNG、WebP
- 用途：展示品牌理念、工艺或产品相关的精美图片

#### 4.2 PEARL STORY 图片  
- 位置：首页"珍珠知识"部分右侧
- 建议尺寸：**800×1000px** (竖版图片)
- 支持格式：JPG、PNG、WebP
- 用途：展示珍珠细节、工艺流程或相关知识的图片

### 5. 操作步骤
1. 点击对应字段的"选择文件"按钮
2. 从您的电脑选择图片
3. 等待上传完成（会显示"上传中..."）
4. 上传成功后会显示预览图
5. **重要：点击右上角的"保存"按钮**

### 6. 替换已有图片
如果您想更换某个图片：
1. 点击预览图右上角的"删除图片"按钮
2. 重新上传新图片
3. 点击"保存"

### 7. 查看效果
- 回到首页：`http://localhost:5173`
- 滚动浏览页面，您会看到：
  - **OUR PHILOSOPHY** 部分显示您上传的第一张图片
  - **PEARL STORY** 部分显示您上传的第二张图片

## 图片建议

### OUR PHILOSOPHY 图片建议内容：
- 品牌工作室环境
- 珍珠首饰制作过程
- 设计师工作场景
- 精致的珍珠首饰特写
- 品牌理念相关的艺术图片

### PEARL STORY 图片建议内容：
- 珍珠原料（贝壳、珍珠层）
- 珍珠光泽特写
- 不同形状的珍珠
- 珍珠表皮纹理
- 珍珠养殖或采集场景

## 技术实现

- **前端**：React + Vite
- **后端**：Flask API
- **数据存储**：页面内容存储在数据库的 `pages` 表中
- **图片存储**：上传的图片保存到服务器 `/uploads` 目录

## 注意事项

✅ **必做事项**：
- 上传图片后必须点击"保存"按钮
- 建议使用高质量、高分辨率的图片
- 图片比例尽量接近 4:5 或 3:4（竖版）

⚠️ **注意事项**：
- 图片文件大小建议不超过 5MB
- 如果没有上传图片，会显示默认的渐变背景
- 图片会自动适应容器大小
- 支持的格式：JPG、PNG、WebP

## 故障排除

### 如果图片没有显示：
1. ✓ 检查是否已点击"保存"按钮
2. ✓ 刷新首页（Ctrl+F5 强制刷新）
3. ✓ 检查浏览器控制台是否有错误信息
4. ✓ 确认图片已成功上传（查看预览）

### 如果上传失败：
1. ✓ 检查图片格式是否为 JPG/PNG/WebP
2. ✓ 检查图片文件大小（建议小于 5MB）
3. ✓ 检查网络连接
4. ✓ 查看服务器日志
5. ✓ 尝试重新登录管理后台
